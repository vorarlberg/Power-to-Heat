# Power-to-Heat

In diesem Projekt wird ein Heizstab (MY-PY AC ELWA 2) mit dem Überschüssigen PV-Strom versorgt. Der Heizstab hängt in einem 120l Puffspeicher im Heizkreis. Dadurch soll erreicht werden, dass der Heizkessel im Sommer komplett aus gelassen werden kann. Das Warmwasser, soll dann aleine vom Heizstab gemacht werden. Dazu wird im Skript eine Warmwasser Sicherstellung eingebaut.
